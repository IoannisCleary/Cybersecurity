<p><span style="font-family:times new roman,times,serif"><span style="font-size:14px">Spam filter is software that&nbsp;filters incoming messages in order to block incoming spam. Messages are&nbsp;classified&nbsp;by the filter based on certain rules and&nbsp;algorithms. If a message is marked as&nbsp;spam, it usually ends up in the junk/spam&nbsp;folder. If&nbsp;a message does not show any signs of being spam or it is from a trusted sender then it is&nbsp;added to the&nbsp;user&rsquo;s inbox folder.</span></span></p>

<p style="text-align:center"><span style="font-family:times new roman,times,serif"><span style="font-size:14px"><img alt="" src="/media/images/spamfilter_extra.png" style="height:150px; width:320px" /></span></span></p>

<p><span style="font-family:times new roman,times,serif"><span style="font-size:14px">According multiple online sources on spam, there are many tricks used in order to fool spam filters into thinking that the message is a legit message and therefore avoid detection. Here are some of the techniques used by spammers:</span></span></p>

<ul>
	<li><span style="font-family:times new roman,times,serif"><span style="font-size:14px">Line breaks at unexpected parts of a message</span></span></li>
	<li><span style="font-family:times new roman,times,serif"><span style="font-size:14px">Adding spaces between characters in words</span></span></li>
	<li><span style="font-family:times new roman,times,serif"><span style="font-size:14px">Replace spaces with a specific character</span></span></li>
	<li><span style="font-family:times new roman,times,serif"><span style="font-size:14px">Use of characters from foreign languages that look like English characters</span></span></li>
	<li><span style="font-family:times new roman,times,serif"><span style="font-size:14px">Switching around some letters in words</span></span></li>
	<li><span style="font-family:times new roman,times,serif"><span style="font-size:14px">Use of HTML code in various was such as encoding characters or adding tiny letters between letters in words, which cannot be noticed by a user.</span></span></li>
	<li><span style="font-family:times new roman,times,serif"><span style="font-size:14px">Use of JavaScript to hide messages</span></span></li>
	<li><span style="font-family:times new roman,times,serif"><span style="font-size:14px">Using multiple &lsquo;zombie&rsquo; computers to send spam since IP addresses can be added to block lists</span></span></li>
	<li><span style="font-family:times new roman,times,serif"><span style="font-size:14px">In order to fool spam filter algorithms more sentences are added&nbsp; in order to pass the algorithms&rsquo; threshold. These additional sentences can be the same colour with the background of a message so that a user does not notice them.</span></span></li>
	<li><span style="font-family:times new roman,times,serif"><span style="font-size:14px">Use IP addresses with neutral or positive reputation</span></span></li>
	<li><span style="font-family:times new roman,times,serif"><span style="font-size:14px">Using tiny URLs instead of full length URLs</span></span></li>
	<li><span style="font-family:times new roman,times,serif"><span style="font-size:14px">Using encoded URLs</span></span></li>
	<li><span style="font-family:times new roman,times,serif"><span style="font-size:14px">Use HTML to fool recipient by adding a malicious link on a legitimate service&rsquo;s URL</span></span></li>
	<li><span style="font-family:times new roman,times,serif"><span style="font-size:14px">Splitting URLs in different parts and showing how the user can attach the parts to get the original URL</span></span></li>
</ul>

<p><span style="font-family:times new roman,times,serif"><span style="font-size:14px">Spammers study how spam filters work so that they can try to exploit their algorithms in order to bypass detection. Spam filters need to be regularly updated with alternative detecting techniques and improved algorithms. As the detection algorithms advance, some problems that can appear are false positive classifications where legitimate messages are marked as spam. Email services such as GMAIL by Google try to improve their algorithms in order to reduce the number of false positive detections by either using familiarity such as adding an e-mail address to an approved senders list (contacts) or allowing users to create their own filters.</span></span></p>

<p><span style="font-family:times new roman,times,serif"><span style="font-size:14px">There are numerous of patterns that filters tend to look at and there are many websites that help legitimate users avoid spam classification by following some rules in order to improve their message&rsquo;s quality score. Some of the rules that are mentioned online are:</span></span></p>

<ul>
	<li><span style="font-family:times new roman,times,serif"><span style="font-size:14px">Do not use of words such as &lsquo;free&rsquo;, &rsquo;urgent&rsquo;, &lsquo;click here&rsquo;, &rsquo;once in a lifetime&rsquo; or characters such as $$, XXX</span></span></li>
	<li><span style="font-family:times new roman,times,serif"><span style="font-size:14px">Do not focus mainly on transactions.</span></span></li>
	<li><span style="font-family:times new roman,times,serif"><span style="font-size:14px">Avoid excessive use of caps lock</span></span></li>
	<li><span style="font-family:times new roman,times,serif"><span style="font-size:14px">Avoid excessive amount of images or links with not enough text</span></span></li>
	<li><span style="font-family:times new roman,times,serif"><span style="font-size:14px">Do not use of bright colours and irregular font size (too small)</span></span></li>
	<li><span style="font-family:times new roman,times,serif"><span style="font-size:14px">Avoid attachment of files of these formats:&nbsp; exe, bat, com etc.</span></span></li>
	<li><span style="font-family:times new roman,times,serif"><span style="font-size:14px">Claiming that is a legitimate e-mail might trigger a filter into thinking that it is spam</span></span></li>
	<li><span style="font-family:times new roman,times,serif"><span style="font-size:14px">Use well known mailers such as GMAIL, Hotmail, Yahoo etc.</span></span></li>
	<li><span style="font-family:times new roman,times,serif"><span style="font-size:14px">Timing : frequent e-mails from an address can be marked as spam</span></span></li>
</ul>
