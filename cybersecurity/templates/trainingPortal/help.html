django-CMS


If you are an admin user you will see a CMS toolbar at the top of the page. In order to use it you will have to click on the Trainin Portal button.

File upload


- Click on Files button on the cms toolbar and then click View/Upload Files
- Click on the blue Upload button and select the file you want to upload
- Go to the file (should appear in the unfiled folder or some other folder)
- click on the file
- Go to the Advanced section
- Copy the url "/filer_public/- something -/.../ -something -/filename.(file extension)"
-- When you want to link to the file just create a link, select protocol "< other >" and add /media in the beginning of the url: "/media/filer_public/-something-/.../-something-/filename.(file extension)" and then click okay


Creating a quiz


In order for you to create a quiz you have to follow these steps:
- Create a quiz category and sub-category [click on the Quiz button then on the Create Category/Sub-category sidebar buttons
- Create questions [click on the Questions button and then select Create True/False / Multi-choice /Essay. Remember each question is linked to a Quiz Category/Subcategory. If you need an existing question you have to replicate it with the new category]
-Create the quiz [Add quiz name, user friendly url and other options. Add the questions that you want it to contain by selection the questions from the less and clicking the add button.]
-- You can make configure the quiz with success or failure messages, single attempt, minimum score for a user to pass etc.


Creating a chapter


- Click on the Chapter button and then the Create Chapter sidebar button
- Enter a unique number greater than 0 and a title.
- The only special characters allowed are : ':','=' and '&'



Creating an announcement


- Click on the Announcements button on the toolbar and then the Create Announcement sidebar button
- Enter a unique number (for that chapter) greater than 0 and a title.
- Enter an announcement entry. (250 characters long)
- Save


Creating an index element


- Click on the Index Elements button on the toolbar and then the Create Index Element sidebar button
- Enter a unique number (for that chapter) greater than 0.
- Enter an index element entry. (250 characters long)
- Save
- This will appear in the home page's grey container.

Creating an exercise


- Click on the Exercise button on the toolbar and then the Create Exercise sidebar button
- Select chapter it belongs to.
- Enter a unique number (for that chapter) greater than 0.
- Enter an a question / task in the question field
- There are some modes that you can select
---- Multiple mode: Use the text field and the CorrectAnswers field. Add a number to each question in the text field and add the ansers in the CorrectAnswers field in the same format.
---- Diagram mode: Upload an image. Enter AvailableAnswers for the diagram and then the CorrectAnswer
---- Further reading mode: Use only the textfield to create an exercise
---- Disable mode: Select this to disable the exercise
- Save


In order to add the exercise to a page go the Exercise button on the toolbar and then the Add to Page sidebar button
- Select the chapter and page it belongs to.
- Select exercises in the appropriate fields. One for the default and other four below (one for each learning style)
-Save



Creating a page (aka section)


- Click on the Page button and then the Create Page sidebar button
- Enter chapter
- Enter a unique number (for that chapter) greater than 0 and a title.
- The only special characters allowed are : ':','=' and '&'
- Four entries:
-- Default entry is displayed if the learningStyle mode is disabled
-- Learnin Style type entries are displayed to users of that learning style if the learningStyle mode is enabled


Creating a profile


- Click on the Profiles button and then the create Profile sidebar button. - Select the user that the profile is associated with - Enter age - Enter learning style type - Enter testing type

Testing mode


- Click on the Testing Mode button and then the Enable/Disable sidebar button -- When enabled users of type A view default page entries only and users of type B view entries normally. This feature also chanes profile pictures depending on the testing type.

Updating/Viewing


- You can view or update anything in the database by clicking on the View sidebar buttons.
